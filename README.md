# Customer Shopping Behavior Analysis

## Objective
Analyze customer purchase behavior using data analytics.

## Tools Used
- Python (Pandas)
- SQL (MySQL)
- Power BI

## Process
1. Data cleaning using Pandas
2. SQL queries for analysis
3. Dashboard creation in Power BI

## Key Insights
- Top selling products
- Customer purchase patterns
- Discount impact on sales

## Conclusion
This project helps businesses make better decisions using data.
